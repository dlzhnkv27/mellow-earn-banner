# Названия сущностей баннера

- `banner-bg` и `banner-container`: одна и та же сущность (один элемент) размером `1130x200`, с `corner radius 20px`, подготовленная под будущий SVG background.
  - Текущий SVG фон: `./assets/banner-container-bg.svg` (источник: `/Users/ilya/Desktop/Group 4647.svg`).
  - Внутренняя тень: `inset 0 0 0 1px rgba(255, 255, 255, 0.5)` с `mix-blend-mode: overlay`.
- `earn-vault-container-1`: первый внутренний блок в `banner-container`.
- `earn-vault-container-2`: второй внутренний блок в `banner-container`.
- `earn-vault-container`: общий базовый класс для обоих внутренних блоков.
- `vault-1`: белая плашка внутри `earn-vault-container-1`, прижата к левому краю.
- `vault-2`: белая плашка внутри `earn-vault-container-2`, прижата к левому краю.
- `vault-hitbox`: стабильная зона наведения для `vault`, не анимируется и не меняет свои границы при hover.
- `vault`: общий базовый класс для обеих белых плашек.
- `script.js`: логика интерактивного наклона `vault` от позиции курсора.
- `earn-vault-stats`: правый блок статистики внутри `earn-vault-container`, расположен справа от `vault`.
- `earn-vault-stat-row`: строка статистики внутри `earn-vault-stats`.
- `earn-vault-stat-label`: левая колонка строки статистики.
- `earn-vault-stat-value`: правая колонка строки статистики.
- `vault-shapes`: контейнер шейпов внутри `vault`, выровнен по правому краю.
- `vault-circle-shape`: круглый шейп внутри `vault`, подготовлен под будущий SVG background.
- `vault-pill-shape`: черный шейп в форме пилюли справа от круглого шейпа.
- `vault-pill-label`: подпись внутри `vault-pill-shape`, слева от белого круглого контейнера.
- `vault-pill-icon-container`: белый круглый контейнер внутри `vault-pill-shape`, подготовлен под будущую иконку.

# Анимации

- Вне hover у баннера нет постоянных анимаций.
- Общий триггер интерактива: `:hover` на `vault`.
- Технически интерактив привязан к `vault-hitbox`, а анимируется внутренний `vault`, чтобы граница сенсора не двигалась вместе с карточкой.
- У `vault` курсор: `pointer` по всей площади.
- Анимация глубины `vault`:
  - Длительность: `300ms`
  - Тайминг: `cubic-bezier(0.22, 1, 0.36, 1)`
  - Сценарий: `vault` поднимается вверх на `3px`, масштабируется до `1.03`, тень становится более крупной и насыщенной
- Анимация наклона `vault`:
  - Источник: `/Users/ilya/Documents/Vibe/Codex/Mellow/Banner/script.js`
  - Длительность перехода: `300ms`
  - Максимальный наклон: `0deg`
  - Логика: система расчета наклона сохранена, но при текущем значении `0deg` визуальный наклон отсутствует
  - Направление наклона готово следовать за положением курсора по обеим осям, если значение будет увеличено позже
  - Расчет ведется от стабильной геометрии `vault-hitbox`, а не от трансформированного прямоугольника самого `vault`
  - Обновление наклона ограничено одним кадром через `requestAnimationFrame`
  - При выходе курсора из визуального footprint поднятого `vault` наклон возвращается к `0deg`
- Анимация стрелки в `vault-pill-icon-container`:
  - Длительность: `300ms`
  - Тайминг: `ease-out`
  - Количество циклов: `1`
  - Сценарий: текущая стрелка уходит вправо, новая стрелка приходит слева
  - Белый круг `vault-pill-icon-container` работает как маска за счет `overflow: hidden`
- Анимация блика в `vault-pill-shape`:
  - Длительность: `800ms`
  - Тайминг: `ease-out`
  - Количество циклов: `1`
  - Форма блика: круглый шейп `72x72`
  - Градиент блика: горизонтальный линейный, слева `0%` белого, справа `15%` белого
  - Траектория: стартует у левого края пилюли, проходит слева направо по всей пилюле и полностью уходит за правую границу
- Стабилизация hover на границе:
  - Hover не сбрасывается мгновенно в первой пограничной точке
  - Для выхода используется расширенная зона, соответствующая lift и scale карточки
  - Это устраняет цикл `enter -> leave -> enter` на краях приподнятого `vault`
- При `prefers-reduced-motion: reduce` отключаются hover-анимации глубины, наклона, стрелки и блика.
